trait Attributes

This trait adds attribute getter, setters and also a useful fill method that can be used with $fillable and $guarded properties to make sure that only the correct attributes will be set.

It is supposed to be used in model classes in general

Properties

protected array $attributes The model's attributes.
protected array $original The model attribute's original state.
protected array $fillable Once you put at least one string in this array, only the attributes specified here will be changed with the setAttributes method.
protected array $guarded The attributes that are not mass assignable. The opposite to the fillable array;.
bool $mutable Check if model should mutate attributes checking the existence of a specific method on model class. Default is true.
