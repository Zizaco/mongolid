SchemaMapper

class SchemaMapper

The SchemaMapper will map an object or an array of data to a Schema object.

When instantiating a SchemaMapper you should provide a Schema. When calling 'map' the Schema provided will be used to format the data to the correct format.

This class is meant to do the opposite of the EntityAssembler
