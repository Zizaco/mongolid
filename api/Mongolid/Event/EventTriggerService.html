mixed fire(string $event, mixed $payload, bool $halt = false)

Triggers / Dispatches a new event to the registered event handlers if they have been registered.

Parameters

string $event identification of the event
mixed $payload data that is going to be sent to the event handler
bool $halt The output of the event handler will be used in a conditional inside the context of where the event is being fired. This means that, if the event handler returns false, it will probably stop the action being executed, for example, "saving".

Return Value

mixed Event handler return. The importance of this return is determined by $halt
